# Спецификация взаимодействия со сторонниим сервисами

---

> ***Примечание:**
Для приложения необходимо взаимодействие с osu! API, спецификация которого представлена ниже.*

### https://osu.ppy.sh/api/

# Обзор

## **/api/get_user_best**

**URL**

```markdown
**/api/get_user_best**
```

### **Описание**

Получить лучшие результаты указанного пользователя.

### Параметры

- `k`- ключ API **(required)**.
- `u` - user_id или username пользователя **(required)**.
- `m` - режим (0 = osu!, 1 = Taiko, 2 = CtB, 3 = osu!mania).
- `limit` - количество результатов (между 1 и 100).

### Ответ

```json
[{
        "beatmap_id"   : "1582674",
        "score_id"     : "2791309545",
        "score"        : "5320058",
        "maxcombo"     : "472",
        "count50"      : "0",
        "count100"     : "28",
        "count300"     : "325",
        "countmiss"    : "0",
        "countkatu"    : "21",
        "countgeki"    : "64",
        "perfect"      : "0",          // 1 = maximum combo of map reached, 0 otherwise
        "enabled_mods" : "64",         // bitwise flag representation of mods used. see reference
        "user_id"      : "3117824",
        "date"         : "2019-04-27 13:52:15", // in UTC
        "rank"         : "S",
        "pp"           : "467.557"      // float value, 4 decimals
        "replay_available" : "1"        // 1 = replay is available for download, 0 = replay is unavailable

}, { ... }, ...]
```

## **/api/get_user_recent**

**URL**

```markdown
**/api/get_user_recent**
```

### Параметры
